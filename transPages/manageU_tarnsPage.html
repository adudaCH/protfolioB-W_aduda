'Manage You'

MANAGE-U is a simple web application designed to help users manage tasks effectively. Built with JavaScript and styled using Bootstrap, the app allows users to add, view, and organize tasks seamlessly. Users can input new tasks, which are displayed in an organized list, with the option to mark tasks as either "active" or "completed." The interface, enhanced by Bootstrap, is clean and responsive, providing an easy-to-navigate experience on both desktop and mobile devices. JavaScript powers the functionality behind task management, ensuring that users can dynamically update their task lists and easily keep track of what still needs to be done versus what has already been completed. Whether it's for personal use or managing work-related to-dos, MANAGE-U offers a simple and effective way to stay organized.
