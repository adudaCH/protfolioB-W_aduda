'localStorage Market'

This is a simple web application that simulates a market where users can add products along with their prices and keep track of them using the browser's localStorage. Built with HTML and styled using Bootstrap, the app offers a clean and responsive interface, making it easy for users to manage their list of products. The application allows users to add new products by entering the product name and price, which are then stored in localStorage for persistence even after the page is refreshed. The added products are displayed in a list format, styled with Bootstrap's table components, providing a clear and organized view. Additionally, users can calculate the total sum of all listed products in real-time, and there is a feature to clear the list entirely if needed. Bootstrap's form controls and layout system enhance the app's usability and appearance, while localStorage ensures that the data remains available for the user's next visit. This project demonstrates how web technologies can be combined to create a practical and interactive tool for managing basic product data.
