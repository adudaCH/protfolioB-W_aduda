'Wizerding Cart'

This project is a simple, wizard-themed shopping cart application inspired by the Harry Potter universe, built using HTML, CSS, JavaScript, Bootstrap, and Font Awesome. Users can browse and add magical items, complete with their prices, to the shopping cart, which is dynamically displayed in a table format. The application features a visually appealing design, enhanced by Bootstrap for a responsive layout and Font Awesome icons to add a touch of magic, such as wands, potions, and other wizardry-themed elements. As users add items to the cart, JavaScript updates the table, showing the item name, price, and quantity, along with a total price calculation. The app provides a smooth user experience, allowing for easy removal of items or updating quantities, and ensures that the cart is updated in real-time. This fun and interactive project combines the practical functionality of a shopping cart with the charm of the Harry Potter universe, offering a delightful experience for fans while learning the fundamentals of web development with these core technologies.
