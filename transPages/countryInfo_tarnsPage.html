'Country Info'

This is a simple web application that allows users to search for country information by entering the capital city. The application fetches data from the REST Countries API and displays relevant details such as the country's name, population, region, languages, and flag. Built using HTML, Bootstrap, and JavaScript, this app combines a clean, responsive design with real-time data fetching to create an easy-to-use platform. Bootstrap enhances the visual layout, providing a modern and mobile-friendly interface that adjusts seamlessly across devices. The application's core functionality is powered by JavaScript, which handles the API requests and dynamically updates the displayed information based on the user's input. The result is a smooth, efficient user experience, perfect for anyone looking to quickly gather information about different countries based on their capital cities.
