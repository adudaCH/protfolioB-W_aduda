'Guess the Number'

This is a simple web-based game where the user has to guess a randomly generated number between 1 and 10. The user has 5 attempts to guess the correct number, with the game providing real-time feedback after each guess, indicating whether the number is too small or too big. Built using HTML, JavaScript, and styled with Bootstrap, the game ensures a smooth and interactive experience. JavaScript handles the core game logic, including generating the random number, tracking the number of guesses, and determining whether the guess is correct or incorrect. Bootstrap is used to create a clean and responsive interface, making the game visually appealing and user-friendly on all devices. If the user guesses correctly within the 5 attempts, a success message is displayed; if they fail to guess the number in 5 tries, they are shown a failure message and prompted to try again. The use of Bootstrap enhances the layout with styled buttons, feedback alerts, and input fields, while JavaScript keeps the gameplay engaging and dynamic by updating the user with each guess. This combination of technologies creates a simple yet entertaining guessing game that can be easily enjoyed on any platform.
