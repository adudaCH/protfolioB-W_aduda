'Shopping List'

This is a simple web application that simulates shopping list where users can add products to the list and keep track of them using a buttyfull list disgne. Built with HTML and styled using Bootstrap, the app offers a clean and responsive interface, making it easy for users to manage their list of products. The application allows users to add new products by entering the product name, The added products are displayed in a list format, styled with Bootstrap's table components, providing a clear and organized view. Bootstrap's form controls and layout system enhance the app's usability and appearance, while javaScript enable showing and adding the products on the list. This project demonstrates how web technologies can be combined to create a practical and interactive tool for managing basic product data.
